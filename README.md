# AI Nav

一站式 AI 生态资源导航页，收录 MCP 目录、Skill 商店、API 中转站、开发工具等 50+ 精选网站。

**在线访问：[ldt471146.github.io/ai-nav](https://ldt471146.github.io/ai-nav/)**

## 功能

- **11 个分类** — AI 对话、API 中转站、MCP 目录、MCP 热门 Server、Skill 商店、开发工具、云服务、邮箱、文档社区、影视、游戏
- **自定义管理** — 添加/编辑/删除网站和分类，数据持久化到 localStorage
- **壁纸系统** — 内置 4 张 ATRI 壁纸，支持自定义图片 URL
- **实时搜索** — 按 `/` 键快速搜索，实时过滤所有卡片
- **导入/导出** — JSON 配置备份，方便迁移和分享
- **自动升级** — 内置数据更新时自动合并，不丢失用户自定义内容

## 技术

纯单文件 HTML + CSS + Vanilla JS，零依赖，双击即可使用。

## 使用

1. 直接打开 `index.html`，或访问上方在线地址
2. 点击卡片跳转对应网站
3. 点击分类的 `+` 按钮添加网站
4. 点击 Header 的 `壁纸` 按钮切换背景
5. 使用 `导出/导入` 备份你的自定义数据
